## NTPサーバってVMで立てているんちゃうんか？
LXC(Linux Containers)と呼ばれるコンテナ技術を用いて、立てています。

通常のdockerコンテナとは異なり、VMのように操作することが可能なのが特徴です。

なので、VMでわざわざ動かすメリットがないようなのは基本的にLXCで動かしています。

## 利用しているソフトウェア
ただのNTPデーモンです。Ubuntu 24.04で以下のコマンドを実行し、
```
apt-get install -y ntp
```
入るものです。

## NTP Poolプロジェクトへの参加
様々なNTPサーバにランダムでアクセスすることを可能にするプロジェクトです。

NTPサーバのスコアが10以上になったものから公開されていきます。
